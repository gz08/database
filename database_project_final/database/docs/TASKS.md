# 开发任务清单（TASKS）

## 一、项目总目标
完成一个基于 **openGauss + Python + Streamlit** 的学生管理系统，并满足数据库课程大作业的全部关键要求，包括：

- 自定义数据库应用系统题目
- openGauss 数据库实现
- 3类角色权限控制
- 增删改查
- 模糊查询
- 数据库文件

## 二、当前项目范围
本项目当前范围控制为以下核心模块：

- 登录与身份识别
- 系统管理员功能
- 教师/教务功能
- 学生功能
- 学生信息管理
- 课程管理
- 选课管理
- 成绩管理
- 查询与模糊查询
- 权限控制

不优先开发复杂扩展功能，如：
- 通知系统
- 消息中心
- 排课算法
- 图表分析大屏

- 审批流
